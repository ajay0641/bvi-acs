# Quote

Displays a single block of quotation text in a classic blockquote style — indented text with a colored left border and italic type.

## Content model

A single-cell table containing the quote text.

| Quote |
| :---- |
| The best way to predict the future is to invent it. |

The quote text supports rich text (bold, italic, links). It is rendered inside a semantic `<blockquote>` element.
